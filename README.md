# AMSPoker v3b

## Description

Poker Game written in Rust/gwg, based on the original I wrote in 1996 with Borland C++/OWL.

![](poker.jpg)

## Dependencies

* [Good Web Game](https://github.com/ggez/good-web-game)


## Build and run

```
cargo build --release

cargo run --release
```
